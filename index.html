Discord Logging Bot

A simple bot to log warnings, kicks, and more to your Discord server

🚀 Setup Instructions

  1. Download the bot: discord-logging-bot.zip
  2. Extract the folder
  3. Install dependencies: npm install
  4. Set your bot token in config.json
  5. Run the bot: node index.js

🛠 Features

  • Warn users with !warn
  • Logs actions to a #staff-logs channel
  • Stores logs in a logs.json file
  • Modular structure – easy to add more commands

📁 File Structure

discord-logging-bot/ ├── index.js ├── config.json ├── logs.json └── commands/ └── warn.js
